# Xiangqi-SwiftUI
Study SwiftUI, try to make a Xiangqi game

### current progress: 
- moving the piece
- check & check mate
- timer
- invalid move toast message

### next step: 
- surrender, asking for draw
- move rules (continuos check, continuos capture...)
- undo moves
- multi-players

### future plan:
- screen transitioning
- view game log
- blind chess
- setting (time, background, pieces...)
- embbed xiangqi engine for game review
- added IAP
